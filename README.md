# System Integrators Simulator

A browser-based pixel-art platformer game built as a brand easter egg for System Integrators Vermont.

Play in browser: https://system-integrators-vermont.github.io/Simulator/

## How to play

- Arrow keys or WASD to move
- - Space or Up arrow to jump
  - - Up/Down to climb ladders
    - - Esc to pause
     
      - Follow an SI Vermont technician through three levels - Rough In, Installation, and Landscape Lighting - collecting tools and lighting up the project as you go.
